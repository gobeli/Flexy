#Flexy

##A Flexbox-based 12 column grid-system

###Media queries

| Small (s)        | Medium (m)       | Large (m)       |
| ---------------- |:----------------:| ---------------:|
| max-width: 480px | max-width: 768px | max-width: 980px|

###use

```html
<div class="row">
	<div class="col-m2"></div>
	<div class="col-m2"></div>
</div>
<div class="row">
	<div class="col-m2 col-offset-m2"></div>
</div>
```
