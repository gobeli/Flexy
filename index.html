Flexy

A flexbox based 12 column grid-system

check it out

Welcome to Flexy, the super lean flexbox grid-system.

User the mixins or as shown below or get started by initializing the default grid (@include defaultGrid());

Below you can see how you can use the mixins/ the default Grid

Get startet by downloading the files from Github or with bower: bower install flexy-12col-grid

After you've downloaded the latest version either get the compiled version or the _flexy.scss into your Project. If you choose the latter you firstly need to define the breakpoints i.e. @include set-breakpoints(("xs": "0px","s": "480px","m": "768px","l": "980px"));. If you're a bit lazy like me just use the defaultGrid() mixin (@include defaultGrid();) or check out all the other mixins below
